# 记账小系统 — 设计规格说明

> 日期：2026-05-30 | 状态：已确认

## 1. 概述

一个轻量级个人记账 Web 应用，纯前端 + 浏览器本地存储，桌面端使用。核心功能：收支记录录入 + 分类统计。

### 目标用户

个人使用，桌面 Web 浏览器。

### 核心原则

- **极简** — 没有多余功能，严格遵循 YAGNI
- **离线** — 所有数据存储在 localStorage，无需后端
- **直观** — 表单录入流畅，统计一目了然

---

## 2. 功能范围

### 在范围内

| 功能 | 描述 |
|------|------|
| 收支记录 | 手动表单录入：类型、金额、分类、日期、备注 |
| 流水列表 | 查看所有记录，支持删除 |
| 分类管理 | 预设 13 个分类，用户可自由增删改 |
| 月度统计 | 收入/支出/结余卡片 + 分类饼图 + 趋势柱状图 |
| 月份切换 | 统计页可按月份切换查看 |

### 不在范围内（YAGNI）

- 多账户、预算管理、数据导出/导入
- 跨设备同步、周期性记账
- 用户系统、多人协同

---

## 3. 架构

### 技术栈

| 层面 | 选择 |
|------|------|
| 框架 | Vanilla JS（无框架） |

## 4. 数据模型

### Transaction（交易记录）

```typescript
{
  id: string,           // crypto.randomUUID() 或 Date.now().toString(36)
  type: 'expense' | 'income',
  amount: number,        // 正数，如 35.00
  categoryId: string,
  note: string,          // 可选，最长 100 字符
  date: string,          // "YYYY-MM-DD"
  createdAt: string      // ISO 8601
}
```

### Category（分类）

```typescript
{
  id: string,
  name: string,          // 如 "餐饮"
  icon: string,          // emoji，如 "🍔"
  type: 'expense' | 'income',
  isPreset: boolean
}
```

### localStorage 键

| Key | 值 |
|-----|-----|
| `bk_transactions` | `Transaction[]` |
| `bk_categories` | `Category[]` |

### 预设分类

**支出：** 🍔 餐饮、🚇 交通、🛒 购物、🏠 住房、🎮 娱乐、💊 医疗、📚 教育、✨ 其他支出

**收入：** 💰 工资、🎁 礼金、📈 理财、💼 兼职、✨ 其他收入

## 5. 数据流

```
用户操作 → UI 模块 → 逻辑模块 (transactions.js / categories.js)
                         │
                    storage.js (JSON 序列化)
                         │
                    localStorage
                         │
                    storage.js (JSON 反序列化)
                         │
                    UI 模块重渲染
```

- 所有状态变化后，立即持久化到 localStorage，然后重新渲染
- 无事件总线，模块间通过 `app.js` 协调
- `app.js` 维护当前月份状态，stats 页据此过滤数据

## 6. 边界情况与错误处理

| 场景 | 处理 |
|------|------|
| 无记录 | 记账页列表显示空状态插图 + "记下第一笔吧" 引导 |
| 统计无数据 | 饼图/柱状图区域显示 "暂无数据"，不报错 |
| 删除含记录的分类 | 确认对话框，提示将该分类下记录归入 "其他" |
| 金额输入非法 | 输入框限制数字格式，提交时校验 > 0 |
| localStorage 满 | 提示用户导出备份后清理（极少发生，~5MB 限额） |
| 删除记录 | 直接删除，无需确认 |

## 7. 设计风格

- **色彩：** 收入绿色 (`#2ecc71`)，支出红色 (`#e74c3c`)，结余蓝色 (`#1976d2`)
- **整体风格：** 简洁干净，白色背景，圆角卡片，柔和阴影
- **字体：** 系统默认字体栈（`-apple-system, ...`）
- **响应式：** 桌面优先（目标宽度 960px ~ 1200px）
